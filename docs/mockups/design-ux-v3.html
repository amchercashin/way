Графики внизу
На всех трёх экранах график — последний элемент. Контент (карточки/список/поля) — сначала, аналитика — ниже.
CAGR по центру
Крупнее (13px, bold), по центру над столбцами. Хорошо читается, не теряется в углу.
Ожид. дивиденд
Перенесён вниз, между полями и графиком. Логический порядок: данные → прогноз → история.
Порядок элементов на всех экранах одинаковый:
Заголовок → Stat-блоки → Контент (карточки / список / поля) → Прогноз (если есть) → График с CAGR
