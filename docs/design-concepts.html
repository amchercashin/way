Wabi-Sabi
Красота несовершенства и пустоты
Дзен-минимализм, где пустое пространство — это дисциплина. Иероглиф 道 (Путь) едва проступает за цифрами дохода — как философия, которая стоит за каждым числом. Цвета камня и старой бумаги. Каждый пиксель отвоёван у пустоты — и потому каждый на своём месте.

Метафора: Камни в дзен-саду — мало элементов, но каждый расположен с предельной точностью. Твой портфель — тот же сад. Не количество активов, а качество каждого решения.

Ключевые решения

философия
Иероглиф 道 как watermark за главной цифрой — философия за числами. Пустота вокруг элементов = дисциплина на Пути.
типографика
Cormorant Garamond для сумм (утончённость), IBM Plex Mono для меток (точность самурая). Контраст старого и нового.
палитра
Камень, песок, старая бумага. Никакого неона — только тёплые земляные тона. Акцент — потёртое золото, как у старинного свитка.
анимации
Медленные fade-in при появлении (0.8s). Категории раскрываются как разворачивающийся свиток — сверху вниз с задержкой. Числа «проявляются» из тумана.
настроение
Тихий храм в горах. Ты открываешь приложение — и суета рынка стихает. Только твой Путь и твои числа.
02 / 03
Bushido
Реестр воина — кровь и сталь
Приложение как боевой журнал самурая. Каждая выплата — это победа. Каждый актив — оружие в арсенале. Кроваво-красный акцент — не для красоты, а как напоминание: рынок не прощает слабости. Круг энсо (円相) обрамляет главную цифру — доход как центр дисциплины.

Метафора: Ты — ронин на финансовых рынках. Без сюзерена, без страховки. Только твоя дисциплина и твой клинок (портфель), который ты точишь каждый день.

Ключевые решения

философия
Энсо (незамкнутый круг) вокруг дохода — символ незавершённости пути. Разрыв в круге = ты всегда в движении, совершенствование не заканчивается.
типографика
EB Garamond для логотипа и сумм — как шрифт из исторической хроники. IBM Plex Mono для данных — точность и контроль.
палитра
Глубокий индиго-чёрный (#0a0a10) + кроваво-красный (#b8413a). Красный — не декорация, а предупреждение: рынок требует уважения.
структура
«Арсенал» вместо «Категории». Статистика в сетке с hair-line разделителями — как таблица в военном журнале.
настроение
Тренировочный зал додзё перед рассветом. Тишина, сосредоточенность, красный акцент как вспышка катаны. Никакого уюта — только путь воина.
03 / 03
Kintsugi
Золотые швы — сила в трещинах
Кинцуги — японское искусство чинить разбитую керамику золотом. Трещины не прячут — их подчёркивают. Каждый кризис, каждая просадка портфеля — это трещина, которую ты заливаешь золотом дисциплины. Путь визуализирован как цепочка точек — каждая выплата приближает к цели.

Метафора: Твой портфель — чаша, которую рынок разбивает снова и снова. Но ты собираешь осколки и скрепляешь их золотом. Каждый шрам делает чашу дороже.

Ключевые решения

философия
Цепочка точек наверху — визуализация Пути. Яркие = пройденные выплаты, тусклые = впереди. Золотые линии-разделители — швы кинцуги, скрепляющие разделы.
типографика
Zen Kaku Gothic для навигации (современная Япония), Cormorant Garamond для чисел (утончённость). Кандзи вместо иконок категорий: 株 (акция), 債 (облигация), 預 (вклад).
палитра
Почти чёрный (#0b0b0e) + кинцуги-золото (#d4a843). Золото используется точечно — только для сумм дохода и декоративных швов. Всё остальное — оттенки пепла.
микро-детали
Золотые линии-разделители имеют «утолщения» — имитация настоящих швов кинцуги. Цитата внизу меняется каждый день — мудрость для Пути.
настроение
Тёмная чайная комната. Свет золотых швов на старой керамике. Ощущение, что каждая трещина — это опыт, а не потеря. Спокойная сила.

Сравнение концепций

Wabi-Sabi

  • Максимальный минимализм, тишина
  • Иероглиф 道 как фоновый водяной знак
  • Тёплые земляные тона (камень, песок)
  • Медленные, медитативные анимации
  • Для тех, кто ценит покой и созерцательность
  • Минимум UI-элементов, максимум «воздуха»

Bushido

  • Энсо (圓相) как рамка для главной цифры
  • Кровавый акцент — рынок не прощает
  • «Арсенал» вместо категорий
  • Сетка статистики как военный реестр
  • Для тех, кто видит себя воином на рынке
  • Наиболее драматичная, напряжённая эстетика

Kintsugi

  • Золотые швы как визуальная метафора
  • Путь из точек — прогресс выплат
  • Кандзи вместо иконок категорий
  • Ежедневная цитата-напоминание
  • Для тех, кто превращает трудности в силу
  • Баланс между теплом и дисциплиной
